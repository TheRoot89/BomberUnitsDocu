8 Definitions
9 -----------
10 
11 * *Client*: An instance of the game, which can join a room and always has at least one (Main) Player.
12 * *Player: Game representation of one physical human/bot player. Has a player slot in menus and a player
13  figure with controller in a match. Each Player can be customized.
14 * *Main Player*: Each Client needs to have at least one Player, which is set up and loaded during startup and
15  cannot be removed. This is the Main Player.
16 * *Host*: One Client acts as the host of a room, giving him settings options for matches etc.
17  The Host is the Client who created the room and to whom the other Clients joined. This is called "Master
18  Client" in Photon.
19 
20 Hosting
21 -------
22 
23 1. Upon starting the game, a new room containing only the Host as Client is created.
24 2. The Main Player of the Host is created.
25 3. The Main Player is added to Player management.

27 Similarities between Host and Clients
28 -------------
29 
30 This is an important part from an implementation perspective:
31 
32 * Local players and remote Players are displayed in Slots, with the local players coming first and displayed
33  bigger.
34 * An "add Player" button is displayed if new players can still be added. The maximum amount of local and total
35  players thus has to be the same across Clients!
