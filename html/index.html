BomberUnits

Bam! (Pre-alpha footage)

BomberUnits is a Bomberman-alike video game using the Unity Engine. It is intended as a learning project for the various tasks encountered in game development. It is developed in 3D and supports mixed local and remote multiplayer via Photon.

Current master version:

State: Playable with the basic features, as well as online MP!
Unity version to use: 2019.3.15f1
How to start: Check out the correct commit, install and assign the correct Unity version to the project in UnityHub, load the project, load the "MainMenuScene" and click Play. For Multiplayer, you may have to start the Photon Setup first: Window->Photon Unity Networking->PUN Wizard->Set App ID to '7d8d867e-5a63-45d9-ae10-1b5b20070d12'.
