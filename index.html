BomberUnits

BomberUnits is a Bomberman clone in Unity as a learning project. It is developed in 3D and supports mixed local and remote multiplayer via Photon.

Current master version:

State: Playable without customization and online Multiplayer.
Unity version to use: 2019.1.6f1

Last "stable" release:

Tag & Name: v0.2 "LocalMP (Refactored & game over available)"
Hash: 15bf755210fcda3cccf0bd1607e42fce49cc882a
Unity version to use: 2019.1.6f1
How to start: Check out the correct commit, install and assign the correct Unity version to the project in UnityHub, load the project, load the "MainMenuScene" and click Play.
